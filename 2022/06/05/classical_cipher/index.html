古典密码
6月 5, 2022

  • 隐写密码

    • 文字
      • 信息可能隐藏在
        • 乱码中的有意义文本
        • 首字母
        • 对角线（第n行取第n个字母）
        • 不一样的字体
        • 断句
        • 字数、句子数、段数
        • 拼写错误
        • ……
      • 编码方式
        • unicode
        • 电码
        • 培根密码
          常常利用大写/小写代表A/B。
        • ……
    • 图片
      • 与背景颜色相似
      • LSB
    • 音频
      • 音阶
      • 包含电码
      • 频谱
    • 所有文件
      • 文件名
      • 元数据
      • 多文件拼合

  • 移位密码

    • 随机移位
      • 易序词
        （把它写在古典密码里了，但其实这是一个十分常见的手法
      • 用其他信息暗示排序
        比如给了一串数字、利用彩虹颜色顺序等
    • 栅栏密码
    • 倒序
    • 树的遍历

  • 替换密码

    • 单表替换（字母表和单表替换本质相同，故也放在这里）
      • 英文字母表
        这个谁都会吧……
      • 自定义字母表
        当年阿拉伯人用词频破解，现在可以用计算机暴力破解
      • 凯撒密码
        将每个字母按字母表顺序平移若干位。
        变种：汉字凯撒，按unicode码表顺序平移
      • 埃特巴什密码
        A替换为Z，B替换为Y……
      • 猪圈密码 pigpen 经常靠猪（pig）和笔（pen）来暗示
      • 仿射密码
      • 9键 T9
        特征是偶数位不超过4
      • 五笔
      • 键盘
        • 按键盘顺序。
          例：QWER → ABCD
        • 字母上方的数字以及对应行号。
          例：Q → 11，V → 43
        • V形。
          例，s左上和右上的数字为24，所以s加密为24，同理，b → 58，l →9-
        • 数字上的标点。! → 1、@ → 2
        • 变种键盘
      • Wingdings字体
      • 旗语
        每个元素都包含两个方向，可以考虑旗语
      • 国际信号旗
      • 盲文
        每个字由6位二进制组成，题干中暗示“看不见”时，可以考虑盲文的可能性
      • 电码

        • 摩(尔)斯电码 Morse code
          字母不超过4个符号，数字不超过5个符号。
          时长：

          1. 点（·）：1
          2. 划（-）：111 （划的长度是点的三倍）
          3. 字符内部的停顿（在点和划之间）：0
          4. 字符之间的停顿：000
          5. 单词之间的停顿：0000000

        • 博多电码 Baudot code

        • 北约音标字母 NATO phonetic alphabet

        • 任何有序集合……

    • 多表替换

      • 维吉尼亚
        说到需要密钥的加密方式，首先想到的就是维吉尼亚

      • 棋盘/方阵
        所有利用这个方阵的密码都被放到这里了

        1/A2/D3/F4/G5/X
        1/AABCDE
        2/DFGHI/JK
        3/FLMNOP
        4/GQRSTU
        5/XVWXYZ

        • 棋盘密码 Polybius
        • ADFGX密码
        • 四方密码

  • 计算

    • 与、或、非、异或
      不仅限于数字
    • 集合的交集、并集、差集
